## 📌 Project Overview

The **Employment Application Review System (EARS)** is a recruitment management system that simplifies and automates the job application review process.
It allows candidates to apply for jobs online while recruiters can efficiently manage, review, and shortlist applicants.

The system reduces manual work, improves hiring efficiency, and stores applicant data in a centralized database.

---

## ✨ Features

✔ Candidate Registration & Login
✔ Job Application Submission
✔ Resume Upload
✔ Application Review by Admin
✔ Candidate Shortlisting
✔ Email Notifications
✔ Centralized Applicant Database
✔ Job Posting & Recruitment Management

---

## 🧑‍💻 System Modules

### 👤 Candidate

* Register and Login
* View Job Openings
* Apply for Jobs
* Upload Resume

### 🛠 Admin

* Manage Job Posts
* Review Applications
* Shortlist Candidates
* Send Notifications

### 🏢 Employer / Recruiter

* View Candidate Details
* Evaluate Applications
* Schedule Interviews
